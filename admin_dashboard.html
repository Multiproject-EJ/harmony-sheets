Workspace quick reference

Product editing guide

Choose a product to edit. Select a row from the catalog or create something new. All edits are stored locally until you download the JSON file.

Editor header

  • Editing — Mode indicator shown while modifying a product.
  • Untitled product — Default title before a product is selected.
  • Edit another product — Return to the product list without saving.
  • View product page — Open the customer-facing product page in a new tab.
  • Delete — Remove the selected product from the workspace.
  • Save changes — Persist updates to your local workspace.

Field reference

Product ID
Used in URLs. Lowercase and hyphenated.
Name
Main product title.
Tagline
Short descriptive subtitle.
Price
Display price. Example: $29.
Life areas
Comma separated list. Controls product filtering. Example: career, health, fun.
Labels
Comma separated labels displayed as badges. Example: One-time purchase, Instant download.
Description
Long-form body copy. HTML allowed.
Key features
List one feature per line.
What's included
List one item per line.
Gallery
One per line. Use path | alt text. Example: assets/image.webp | Alt text.
FAQs
One per line using question | answer. Example: Question? | Answer.
Benefits
One per line using title | description.
Pricing title
Optional heading for pricing details.
Pricing subtitle
Secondary text supporting the pricing call-to-action.
Stripe checkout URL
Link to the Stripe checkout page. Example: https://.
Etsy URL
Link to the Etsy listing. Example: https://.
Accent image
Primary supporting image path. Example: assets/example.webp.
Accent caption
Caption or supporting copy for the accent image.
Demo video
Video file path. Example: assets/demo.mp4.
Demo poster
Poster image path for the demo video. Example: assets/demo-poster.jpg.
Social proof stars
Star rating display. Example: ★★★★★.
Social proof quote
Short testimonial. Example: Short testimonial.

Board library controls

Initializes the modal layer that launches each sticky board experience and wires up dismissal logic for overlays, close buttons, and keyboard shortcuts.

Hook Purpose
data-board-layer Root element containing the sticky board modals and overlay.
data-board-modal Identifies each modal instance (for example, brain or flowchart).
data-board-launch Buttons that open the corresponding modal and store the triggering control.
data-board-close & overlay Close affordances that collapse the active modal and restore focus.

Implementation notes

  • Document keyboard focus management and animation hooks.
  • Add overlay styling guidelines if custom themes are introduced.

Brain board workspace

Creates the draggable sticky note canvas for the brain board, registers Supabase save actions, and exposes board switching utilities.

Hook Purpose
data-brain-board Root node passed to the StickyBoard class.
#brain-board-note-template Template element used to clone new sticky notes.
data-board-select / data-board-switcher Dropdown and dialog that load and save Supabase board layouts.
data-board-save & data-board-status Primary save trigger and status region for progress or errors.

Implementation notes

  • Clarify default snapshot expectations before overwriting custom boards.
  • Track additional accessibility announcements for save operations.

Flowchart board workspace

Spins up the flowchart variant of the sticky board with optional initial zoom and delegates updates to the shared prompt compiler.

Hook Purpose
data-flowchart-board Root element instantiated as a StickyBoard instance.
data-board-initial-scale Optional attribute that seeds the default zoom level.
data-board-workspace Scrollable canvas that emits updates consumed by the prompt module.
data-board-add controls Buttons that insert new flowchart sticky notes into the workspace.

Implementation notes

  • Outline swimlane presets once the placeholder controls are implemented.
  • Document syncing behaviour when multiple flowchart boards are opened.

Flowchart prompt compiler

Compiles the live flowchart workspace into a sharable prompt, exposes a manual regenerate action, and provides clipboard helpers.

Hook Purpose
data-flowchart-prompt Container that scopes prompt controls and output fields.
data-flowchart-prompt-generate Button that triggers a manual prompt compilation.
data-flowchart-prompt-copy Invokes clipboard logic or falls back to manual text selection.
data-flowchart-prompt-output Read-only textarea populated with compiled prompt text.

Implementation notes

  • Add localization guidance for success and error status messages.
  • Capture performance metrics for large flowchart boards.
